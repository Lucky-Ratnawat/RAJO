# RAJO

RAJO is a mobile-first B2B commerce platform starting with imitation jewellery and designed to expand into additional wholesale categories later.

## Workspace Layout

```text
apps/
  admin/      React admin dashboard
  mobile/     Expo React Native buyer app
services/
  api/        FastAPI backend
docs/         Product and technical documentation
```

## Current Focus

- open browsing for all users
- login required for ordering
- single-brand B2B wholesale flow for launch
- category-first catalog and inventory management

## Next Build Steps

1. Finish backend domain models and migrations.
2. Add admin CRUD for categories, products, and inventory.
3. Build buyer catalog and order flow in the mobile app.
